foa3d.output¶

foa3d.output.create_save_dirs(img_path, img_name, cli_args, is_fiber=False)¶

Create saving directory.

Parameters

  • img_path (str) – path to input microscopy volume image

  • img_name (str) – name of the input volume image

  • cli_args (see ArgumentParser.parse_args) – updated namespace of command line arguments

  • is_fiber (bool) – True when fiber orientation vectors are provided as input to the pipeline

Returns

save_subdirs – saving subdirectory string paths

Return type

list (dtype=str)

foa3d.output.save_array(fname, save_dir, nd_array, px_sz=None, fmt='tiff', odi=False)¶

Save array to file.

Parameters

  • fname (string) – output filename (without extension)

  • save_dir (string) – saving directory string path

  • nd_array (NumPy memory-map object or HDF5 dataset) – data

  • px_sz (tuple) – pixel size (Z,Y,X) [um]

  • fmt (str) – output format

  • odi (bool) – True when saving ODI maps

Return type

None
